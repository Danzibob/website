Objects

An object is a new type of thing that we can use when programming, just like int || string || float. Objects are created with constructor functions in which we define what we want them to do. A constructor function is different from a normal function in that we write ClassName = function(){...}

To create a new instance of an object, we use the new statement: 

Thing = function(){}
var a = new Thing()

However, in this state objects are pretty useless. Move on to the next tab for some uses

Attributes

An attribute is another variable stored within an instance of an object. We designate it with the keyword this. By giving the constructor function parameters we can assign a value to an attrubute when we create an object: 

Thing = function(newValue){
    this.value = newValue
    this.age = 0
}
var a = new Thing(4)
var b = new Thing(2)
console.log(a.value, b.value)
// Prints 4 , 2

This can be useful for storing position or velocity etc. as shown in the example to the left

Methods

A method is a special function that runs inside an object, and can use and change the attributes of that object. Here, methods are used to simulate all the physics of the ball, inside the ball object. While this may seem like more code than neccessary, it becomes more useful if we have lots of objects of the same type, all of which have different attributes or colors etc.

Here's a simpler example:

Thing = function(name){
    this.name = name
    this.greet = function(){
        console.log("Hello, my name is ", this.name)
    }
}
var a = new Thing("Bob")
var b = new Thing("Jeff")
a.greet()
// Prints: Hello, my name is Bob
b.greet()
// Prints: Hello, my name is Jeff
