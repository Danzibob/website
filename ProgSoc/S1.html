The Canvas

The canvas is the main part of any p5 sketch. It's where everything is drawn. The code you see to the left consists of two parts: setup and draw.

  • setup only runs once - as soon as the page is loaded
  • draw runs 60 times a second, or as fast as it can go below that

The createCanvas function does what it says on the tin - creates a canvas. The two numbers are the X and Y dimensions of the canvas respectively, so this canvas would be 400px wide and 300px tall. By default the canvas is blank. The next step is to add color. Click the tab above to move on.

Color

The way computers display color is through lighting up Red, Green and Blue LEDs in each pixel to produce the desired color. Light mixes differently to the color system you may be used to. Try it out below:

Because of this system, the way we tell a computer a color is with a list of 3 numbers, corresponding to the Red, Geen and Blue, hence RGB. These numbers are usually between 0 and 255, but are also often in hexadecimal (eg #6D2B6A). But in p5 we can stick to normal numbers. For example, the purple we have above would be (109,43,106)
