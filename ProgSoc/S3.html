IF Statements

If statements are the programmer's way of telling the computer to make a decision; do I run this bit of code or not?

if(i < 10){ //IF the variable i is less than 10
//Run this bit of code
}

In a sketch we can use this to determine if a certian condition is met. For example on the left, an ellipse is only drawn when the mouse is in the left half of the canvas

ELSE Statements

The ELSE statement allows us to have an alternative action for when our IF statement isn't run.

if(i < 10){ //IF the variable i is less than 10
//Run this bit of code
} else { //ELSE the variable i ISN'T less than 10
//Run this bit of code instead
}

This time the sketch on the left performs a different action depending on if the mouse is in the left or right half of the canvas

Math

We've already had some basic math, but some of the more useful operations are as follows

a % b Is the modulus division operator - it gives the remainder when dividing a by b. This is useful when you want to keep a value in a certain range, eg mouseX % 255 for a color parameter

constrain(x, minX, maxX, minY, maxY) lets you take a value you know the maximum and munimum values of, and scale them from that range to a custom range
