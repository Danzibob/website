p5.js

"Hello! p5.js is a JavaScript library that starts with the original goal of Processing, to make coding accessible for artists, designers, educators, and beginners, and reinterprets this for today’s web."

p5.js is a library for JavaScript that focuses on the canvas, for visual projects of all kinds to be used on the web. It's intended for beginners to coding, designers and anyone in a creative arts field. The project is open source and anyone can contribute through github.
