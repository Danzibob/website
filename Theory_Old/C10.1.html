The CPU

The CPU is often described as the 'brain' of the computer system. This is rather misleading, but it does carry out all the instructions given to it. Inside a CPU there are billions of transistors (electronic switches), combined into logic gates which can perform arithmetic and logical operations

Moore's Law

Gordon Moore (founder of Intel) predicted in 1965 that the number of transistors on a CPU would double approximately every 2 years. Since then this has held true.

Clock Speed

The clock speed of the CPU is the number of cycles per second. Clock speed is measured in Hertz (Hz). Modern CPUs run in the range of Gigahertz (1,000,000,000 or more cycles per second). The greater the clock speed of a CPU, the better it's performance. However, performance can be affected by many other factors.

Inside the CPU

NB: This model of a CPU is an abstraction; a simplifed version to make it easier to understand. Modern processors are far more complex

Program Counter (PC)

Keeps track of the memory location of the line of code being executed. Is incremented each cycle. Can also be changed by control flow operations (e.g: BRP, BRZ and BRA in the LMC

Memory Data Register (MDR)

Stores data that has been fetched from or stored in memory

Memory Address Register (MAR)

Stores the address of the data or instructions to be fetched from or sent to memory

Current Instruction Register (CIR)

Stores the most recently fetched instruction, waiting to be decoded and executed

Accumulator (ACC)

Stores the results of calculations by the ALU

General Purpose Registers

Memory locations within the CPU which can temporarily store data and instructions instead of shipping them back and forth to the much slower memory locations

Buses

General channels of communication around the CPU. There are 3 main buses to be aware of:

  • Data bus: carries data between the processor and memory
  • Address bus: carries the address of the memory location to be written to or read from
  • Control bus: carries signals from the control unit

Arithmetic Logic Unit (ALU)

Carries out all calculations and logical decisions

Control Unit (CU)

Sends out signals to co-ordinate how the processor works.It controls how data moves around the CPU, and also decodes instructions for the CPU to run.
