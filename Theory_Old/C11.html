Numbers

Numbers in a computer system are stored as bytes of 1s and 0s, using the binary number system:

1286432168421
01101001

Negative Numbers

There are 2 main methods of storing negative numbers in computers: Two's compliment, and sign magnitude

Two's compliment

In this method, the MSB's (Most Significant Bit's) value is made negative, as shown below with the number -92:

-1286432168421
10100100

This means that the maximum value for a two's compliment binary integer is |MSB|-1, and the minimum value is -|MSB|

Sign magnitude

Sign magnitude replaces the MSB with a bit indicating a sign, with 0 meaning positive, and 1 meaning negative. This is demonstrated below, also with -92

+/-6432168421
11011100

The max and min values for sign magnitude numbers are one less than the original MSB
